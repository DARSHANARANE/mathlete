# 📚 Student Management System

A modern web application built using React.js to manage student-related workflows. The platform allows users to browse, purchase, and download question papers, along with an admin panel for managing system data.

---

## 🚀 Features

### 👨‍🎓 Student Portal
- Browse available question papers
- View detailed information
- Add to cart & checkout
- Download purchased papers
- View order history

### 🛠️ Admin Panel
- Secure admin login
- Upload/manage question papers
- Manage users & orders
- System configuration

---

## 🧑‍💻 Tech Stack
- React.js
- TypeScrpit
- Redux (State Management)
- REST APIs

---

## 🚀 Install dependencies:

1) **npm install**
2) **Run development server:**
   
        npm run dev
4) **Build for production:**
   
        npm run build
6) **Preview production build:**
   
        npm run preview
